# Assets 폴더

이 폴더는 README.md에서 사용하는 이미지 파일을 저장합니다.

## 필요한 이미지

### 1. console-output.png
콘솔 출력 스크린샷

**캡처 방법:**
```bash
java -jar build/libs/code-flow-tracer.jar -p samples -s detailed
```
- Windows Terminal에서 위 명령어 실행
- 결과 화면 캡처 (Win + Shift + S)
- `console-output.png`로 저장

### 2. excel-output.png
엑셀 출력 스크린샷

**캡처 방법:**
1. `assets/demo-output.xlsx` 파일 열기
2. "호출 흐름" 시트와 "SQL 목록" 시트 캡처
3. `excel-output.png`로 저장

## 이미지 권장 사항
- 가로 800px 이상 권장
- PNG 형식 사용
- 불필요한 여백 제거
